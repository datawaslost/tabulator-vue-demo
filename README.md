# Tabulator Vue Demo

A blank Vue 3 + Vite project demonstrating Tabulator in a single-page transaction-history table.

## Features

- Column sorting and shortcut sort buttons
- Header filters and global search
- Status filtering
- Pagination, responsive columns, and movable columns
- CSV, JSON, HTML, Excel, and PDF downloads

## Run

```sh
nvm use
npm install
npm run dev
```

The latest Vite release used here requires Node 20.19+ or 22.12+.
